Info Page Accessibility Test

Test Steps

  1. Open browser console (F12)
  2. Navigate to /info in your app
  3. Check console for any errors or logs
  4. Look for these specific log messages:
    • "Info component initialized"
    • "Info page useEffect"
    • "Fetching profile for user"
    • "Profile fetch result"
    • "Info page render"

Expected Behavior

  • Page should load without console errors
  • Should show loading spinner initially
  • Should display gamification info after loading
  • Should show default profile data if user has no profile

Common Issues

  • White screen: Check for JavaScript errors in console
  • Loading forever: Profile fetch might be stuck
  • Redirect loops: Check ApplicationStatusGuard
  • Authentication issues: Check useAuth hook

Debug Commands

Run these in browser console:

// Check if user is authenticated
console.log('User:', window.supabase?.auth?.user);

// Check current route
console.log('Current path:', window.location.pathname);

// Check for React errors
console.log('React DevTools available:', !!window.__REACT_DEVTOOLS_GLOBAL_HOOK__);
